# Travelgram - Perfil de Viagens

Este é um projeto de perfil de viagens, desenvolvido como parte de meus estudos em desenvolvimento web. Ele simula uma página de perfil de um viajante, exibindo informações pessoais, estatísticas de viagem e uma galeria de fotos.

## Tecnologias Utilizadas

*   **HTML5**: Para a estrutura do conteúdo.
*   **CSS3**: Para estilização e responsividade.
*   **Fontes do Google Fonts**: Para tipografia.
*   **Imagens e Ícones Locais**: Para elementos visuais.

## Estrutura do Projeto

*   `index.html`: Página principal com o perfil e a galeria de fotos.
*   `styles/`: Contém os arquivos CSS para estilização (`global.css`, `header.css`, `main.css`, `footer.css`, `nav.css`, `index.css`).
*   `assets/icons/`: Armazena os ícones utilizados na página.
*   `assets/images/`: Contém as imagens da galeria e a foto de perfil.

## Propósito do Estudo

Este projeto serve como um exercício prático para aprimorar minhas habilidades em:
*   Estruturação semântica de HTML.
*   Estilização com CSS, incluindo layout (flexbox/grid) e design responsivo.
*   Organização de arquivos e assets em um projeto web.

Sinta-se à vontade para explorar o código!
